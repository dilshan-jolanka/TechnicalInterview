Timer:

How It Works

The timer is implemented as a standalone HTML component that can be embedded in any page using an iframe. The timer updates continuously without causing any page refresh.

To use in your own applications:

  1. Include the iframe pointing to timer.html
  2. Send a message to the iframe with start time and time limit
  3. Listen for timer_expired messages from the iframe

// Send data to timer
const iframe = document.getElementById('timer-frame');
iframe.contentWindow.postMessage({
    type: 'updateTimer',
    startTime: new Date().toISOString(),
    timeLimit: 300  // seconds
}, '*');

// Listen for timer expiry
window.addEventListener('message', function(event) {
    if (event.data === 'timer_expired') {
        alert('Time expired!');
    }
});
